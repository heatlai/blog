# blog

## install
brew install ruby  
gem install jekyll  
gem install bundler -v 1.12  
bundle _1.12_ install  
bundle install  

## run
bundle exec jekyll build  
bundle exec jekyll serve  
bundle exec jekyll serve --watch  
> => "http://localhost:4000"

bundle exec jekyll serve --host myhost --port 7788
> => "http://myhost:7788"
